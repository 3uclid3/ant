# Contributing

Thanks for your interest in `ant`.

## Current Status

`ant` is in early development and **not accepting pull requests at this time.** The API and internals are actively changing, and outside contributions would be difficult to integrate responsibly before the foundation stabilizes.

## How You Can Help

Feedback and ideas are genuinely welcome:

- **Bug reports:** open an issue if something behaves unexpectedly
- **Design feedback:** if the API feels awkward for your use case, open an issue and describe it
- **Questions:** issues are fine for questions too

## Roadmap

Once the core is stable and the project reaches 1.0, contributions will be opened up properly with guidelines, a code style document, and a clear scope.

## Code Style

`ant` targets **C++26** and is built with **GCC 16+** and **CMake**. If you are forking or experimenting locally, please keep that in mind.
